# Installation Instructions for Personnel Management Application

## Step 1: Install Node.js
Make sure to have Node.js installed on your machine. You can download the installer from [Node.js official website](https://nodejs.org/).

## Step 2: Clone the Repository
Open your terminal or command prompt and execute:
```bash
git clone https://github.com/geomensor2006-coder/PERSONAL-EN-OBRA.git
```

## Step 3: Install Dependencies for Backend
Navigate into the backend directory and install the dependencies:
```bash
cd PERSONAL-EN-OBRA/backend
npm install
```

## Step 4: Install Dependencies for Frontend
Navigate into the frontend directory and install the dependencies:
```bash
cd ../frontend
npm install
```

## Step 5: Run the Application Locally
To start the backend server, navigate back to the backend directory and execute:
```bash
cd ../backend
npm start
```

To start the frontend application, navigate to the frontend directory and execute:
```bash
cd ../frontend
npm start
```

After starting both servers, you can access the application in your web browser at `http://localhost:3000`.
